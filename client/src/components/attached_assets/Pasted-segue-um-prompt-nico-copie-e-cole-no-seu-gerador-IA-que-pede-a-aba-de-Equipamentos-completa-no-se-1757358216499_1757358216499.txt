Regras de status/severidade

vencido se prazo < hoje e status !== concluido

alta se tipo = descarte vencido OU calibracao vencida OU manutencao crítica; media se vence em ≤ 30 dias; baixa caso contrário; info para concluídos.

Badge de cor: alta (vermelho), média (amarelo), baixa (azul), info (cinza).

Campos específicos por tipo (em meta)

manutencao: { ordemServico, tipo: "preventiva"|"corretiva", checklist?: string[] }

garantia: { fornecedor, slaHoras, fimGarantia }

software_licenca: { produto, chaveUlt4, validade }
