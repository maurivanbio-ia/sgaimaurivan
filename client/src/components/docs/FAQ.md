# ❓ FAQ - Perguntas Frequentes

## 🚀 Geral

### O que é o EcoBrasil Gestor?

É uma plataforma completa de gestão ambiental e licenciamento com IA integrada, desenvolvida para otimizar processos de compliance e gestão de empreendimentos.

### Quais unidades são suportadas?

Atualmente:
- Goiânia (GO)
- Salvador (BA)
- Luiz Eduardo Magalhães (BA)

---

## 🔐 Autenticação

### Como faço login?

1. Acesse `/login`
2. Use seu email corporativo
3. Senha é a mesma do sistema anterior

### Esqueci minha senha

Contact e o administrador do sistema para reset.

### Posso ter acesso a múltiplas unidades?

Sim! Usuários admin podem alternar entre unidades em "Selecionar Unidade".

---

## 📊 Dashboard

### Os dados são em tempo real?

As estatísticas são **cacheadas por 5 minutos** para performance. Dados críticos são sempre atualizados.

### Como export ar relatórios?

Use o botão "Exportar" no dashboard e escolha PDF ou Excel.

---

## 🤖 Assistente IA

### Como usar o EcoGestor-AI?

1. Click no ícone de IA
2. Digite sua pergunta
3. A IA analisa seus dados e responde

### O que a IA pode fazer?

- Consultar dados (licenças, veículos, etc)
- Analisar documentos
- Fornecer insights
- Executar ações (listar, buscar)

### A IA tem memória?

Sim! A IA lembra das **últimas 6 mensagens** para contexto.

### Quanto custa usar a IA?

Custo médio: **R$ 0,02 por consulta**. Veja métricas em Analytics.

---

## 📁 Dados

### Onde ficam meus arquivos?

Todos os uploads são armazenados de forma segura no servidor com validação anti-malware.

### Posso recuperar um registro deletado?

Sim! **Soft delete** permite restore dentro de 90 dias. Contate o admin.

### Como funciona o backup?

Backup **automático diário** às 2h da manhã. Retenção: 30 dias.

---

## 🔔 Notificações

### Como ativar notificações push?

1. Vá em Configurações
2. Click em "Ativar Notificações"
3. Permitir no browser

### Que notificações recebo?

- Licenças vencendo
- Entregas pendentes
- Demandas urgentes
- Alertas do sistema

## ⚡ Performance

### O site está lento

1. Limpe cache do browser (Ctrl+Shift+Del)
2. Verifique conexão internet
3. Tente outro browser

### Dashboard demora muito

Primeira carga pode demorar ~2s. Depois, cache acelera para ~200ms.

## 🛠️ Troubleshooting

### Erro "Não autorizado"

Faça login novamente. Sessão expira após inatividade.

### Erro "Limite de requisições"

Rate limit: 100 req/min. Aguarde 1 minuto e tente novamente.

### Upload falha

Verifique:
- Arquivo < 10MB
- Formato permitido (PDF, DOC, XLS, IMG)
- Conexão estável
