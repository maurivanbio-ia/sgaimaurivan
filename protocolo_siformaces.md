# PROTOCOLO DE COLETA DE INFORMAÇÕES — SIFORMACES / ECOGESTOR
**Ecobrasil Consultoria Ambiental**
Versão 1.0 | Uso interno

---

> **Objetivo:** Este protocolo orienta a coleta de todas as informações necessárias para o cadastro completo dos projetos no sistema EcoGestor, garantindo que nenhum dado relevante seja omitido.
> 
> Para cada projeto, preencha todas as seções aplicáveis. Campos marcados com **(!)** são obrigatórios.
