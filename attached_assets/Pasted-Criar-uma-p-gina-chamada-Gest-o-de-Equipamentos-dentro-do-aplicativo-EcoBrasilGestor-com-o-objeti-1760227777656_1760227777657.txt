Criar uma página chamada “Gestão de Equipamentos” dentro do aplicativo EcoBrasilGestor, com o objetivo de listar, filtrar, cadastrar e gerenciar equipamentos ambientais e operacionais utilizados nos projetos da empresa.

A página deve integrar-se visualmente com o restante do sistema (mesma navbar, tema, tipografia e layout).

🧭 Localização no Projeto

Caminho sugerido no Next.js App Router:

/app/equipamentos/page.tsx

🧩 Componentes a utilizar

Usar os componentes padrão do sistema, vindos de:

@/components/ui/ (botão, input, card, tabela)

@/components/custom/ (Header, Sidebar)

🧠 Requisitos Funcionais
1. Listagem de Equipamentos

Exibir uma tabela com as seguintes colunas:

ID

Nome do Equipamento

Tipo (Ex: Veículo, GPS, Drone, Armadilha, Estação Meteorológica)

Localização Atual

Status (Em uso, Disponível, Em manutenção)

Responsável

Última manutenção

Ações (Editar / Excluir)

✅ A listagem deve ser carregada de um mock JSON inicialmente, e futuramente integrar-se com uma API (endpoint /api/equipamentos).

2. Filtros

Criar campos de filtro na parte superior:

Campo de busca por nome (Input)

Filtro por tipo (Select)

Filtro por status (Select)

Botão “Limpar filtros”

3. Cadastro e Edição

Adicionar um botão “+ Novo Equipamento” no topo da página.
Ao clicar:

Abrir um Dialog (modal) com os seguintes campos:

Nome

Tipo

Localização

Responsável

Status

Data da última manutenção

Botão Salvar e Cancelar

4. Layout visual

Usar Card do shadcn/ui para agrupar a tabela e filtros.

Aplicar flex, gap-4 e p-6 para espaçamento.

Usar cores e fontes do tema global (text-gray-900, bg-gray-50, etc.).

Seguir o mesmo estilo do módulo de “Gestão de Projetos”.
